Proyecto Final de Carrera – Videojuego
Descripción

Este documento establece las bases para el desarrollo del proyecto final de carrera, cuyo objetivo es diseñar y desarrollar un videojuego.
El proyecto contempla la planificación, diseño, desarrollo y documentación completa del videojuego, aplicando metodologías de gestión de proyectos y buenas prácticas de desarrollo de software.

Tipo de juego

El videojuego estará inspirado en alguno de los siguientes géneros:

Metroidvania

Roguelike

Estos géneros se caracterizan por la exploración, progresión del personaje y desafíos constantes dentro del juego.

Tecnologías utilizadas
Lenguaje de programación

C#

Motor gráfico

Unity

Arquitectura técnica

Arquitectura Component-Based (Basada en Componentes)

Este enfoque permite crear objetos del juego mediante componentes reutilizables, facilitando la modularidad y escalabilidad del proyecto.

Metodología de trabajo

Para la gestión del desarrollo se utilizará la metodología ágil:

Scrum

Scrum permitirá organizar el trabajo mediante sprints, reuniones periódicas y seguimiento continuo del progreso del proyecto.

Seguridad del código

Para proteger el software desarrollado se implementarán medidas de seguridad como:

Ofuscación de código

Encriptación

Estas técnicas ayudan a prevenir la ingeniería inversa y proteger la lógica interna del videojuego.

Temáticas posibles del videojuego

El videojuego podrá desarrollarse utilizando una o varias de las siguientes temáticas:

Dinosaurios

Terror cósmico

Espacio

Zombis

Medieval

Dulces

Mitología lovecraftiana

Piratas

La temática final será definida durante la fase de diseño del proyecto.

Entregables del proyecto

Durante el desarrollo del proyecto se generarán los siguientes documentos y artefactos:

Matriz de trazabilidad

Requerimientos de software

Gestión de alcance

EDT (Estructura de Desglose del Trabajo)

Presupuesto del proyecto

Acta de constitución

Control de cambios

Carta Gantt

Casos de uso

Estos entregables permitirán mantener una correcta planificación, seguimiento y control del proyecto.

Objetivo del proyecto

El objetivo principal es aplicar los conocimientos adquiridos durante la carrera para diseñar, desarrollar y documentar un videojuego funcional, utilizando herramientas profesionales de desarrollo y gestión de proyectos.
